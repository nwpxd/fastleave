# FastLeave

Leave Fortnite matches instantly with one key press.

## Setup

1. Download `fastleave.exe` from [Releases](https://github.com/nwpxd/fastleave/releases)
2. Put it anywhere on your PC
3. Run it (right-click > Run as administrator if needed)
4. Open Fortnite
5. Press F6 in a match to leave

That's it.

## How it works

When you press the hotkey, the app automatically:
- Opens the Escape menu
- Clicks the exit icon
- Clicks "Return to lobby"
- Clicks "Yes"

All in about 1 second.

## Settings

- **Set Key** — Change the hotkey (default is F6). Click the button, then press the key you want.
- **Enabled** — Toggle the macro on/off without closing the app.
- **Minimize to tray** — When minimized, the app goes to the system tray. Right-click the tray icon to quit.

## Requirements

- Windows 10/11
- Screen resolution: 1920x1080
- Fortnite in fullscreen or borderless windowed

## FAQ

**Nothing happens when I press the key.**
Make sure the app says "READY" and the hotkey isn't used by another program.

**It clicks the wrong things.**
Your resolution must be 1920x1080. Other resolutions are not supported.

**The app won't start.**
Right-click > Run as administrator.

**How do I close it?**
Close the window, or right-click the tray icon > Quit.
